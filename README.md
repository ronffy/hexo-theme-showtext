## 怎样使用本主题

```bash
# 在hexo项目执行如下命令
cd themes
git clone https://github.com/ronffy/hexo-theme-showtext.git
```

```yml
# 修改你hexo项目的_config.yml
theme: hexo-theme-showtext
```

更多配置项，请参照[DEMO](https://github.com/ronffy/hexo-roronoa-blog)
